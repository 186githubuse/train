# 杨老师感觉训练营

面向 1-12 年级学生的写作感知力培养 App，通过 10 节课 + 专题训练帮助学生掌握"用感觉写作文"的方法。

## 功能现状

- **基础训练**：10 节感觉训练课程，配套分难度题库、错题本、挑战赛、学习报告。
- **专题训练**：已开放静物 / 植物 / 动物 / 景物 4 类，共 13 个子内容，统一采用 `schema:'unit'` 单元模块化「边学边写」流程。
  - 静物：台灯、笔袋、书包
  - 植物：柳树、白牡丹、橘子
  - 动物：橘猫、比熊犬·小V、虎皮鹦鹉、金鱼、蜜蜂
  - 景物：状元桥、公园湖畔
- **AI 能力**：专题分段书写 / 综合书写评分、手写图片 OCR、TTS 语音与音效反馈。
- **图片状态**：蜜蜂、状元桥、公园湖畔正式主图已上传 COS 并接入；8 张新增局部图也已通过 `imageOverride` 接到相关选择题。

## 快速开始

本项目是纯原生 Web 应用，无需安装依赖。

### 本地运行

```bash
# 必须用 HTTP 服务器（不支持 file:// 协议）
python3 -m http.server 8080

# 浏览器访问
open http://localhost:8080
```

### 最佳体验

- **PC/iPad 横屏**：训练营采用2列网格布局，专题训练3列网格，充分利用横屏空间
- **移动端**：竖向地图布局，适配单手操作
- **浏览器要求**：Chrome/Safari/Edge 现代浏览器

### ⚠️ 重要：API 密钥配置

项目需要 `js/config.js` 文件（包含 API 密钥，**未提交到 git**）。

首次运行需手动创建该文件，格式参考项目文档。

## 技术栈

- 纯原生 HTML / CSS / JavaScript (ES Module)
- Tailwind CSS (CDN)
- localStorage 持久化
- 无框架、无构建工具

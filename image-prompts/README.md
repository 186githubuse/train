## 一、统一风格规范（所有图共用）

### 视觉风格
- **风格基调**：日系软插画 + 现代极简静物摄影的混合体
- **质感**：写实为底（物体比例、结构、色彩准确）+ 轻度风格化（柔和边缘、淡淡颗粒感、降低饱和度到 80%）
- **不要**：3D 卡通 / 油画 / 赛博朋克 / 商业摄影硬光 / 任何文字、logo、品牌

### 色调
- **背景色**：`#F8F0E3` 奶油色，略带温暖米色
- **物体色**：保留各自客观色彩（深蓝就是深蓝、棕黑就是棕黑），但整体饱和度降低 20%
- **光线**：左上 45° 单点柔光（自然晨光），淡阴影

### 情绪关键词
`温馨` `舒适` `专注` `童趣` `有温度` `清晨自然光` `桌面静谧感`

### 构图通则
- 主体居中或黄金分割点
- **至少 30% 负空间**留白（方便 App UI 叠加）
- 浅景深，背景虚化
- 不带任何文字、贴纸、品牌标识

### 分辨率与比例
| 图类 | 比例 | 像素 |
|---|---|---|
| 静物主图 | 16:9 横版 | 1920 × 1080 |
| 树状图 | 1:1 方形 | 1024 × 1024 |
| 局部特征图 | 1:1 方形 | 1024 × 1024 |

> **重要提醒**：AI 生成的中文文字几乎一定是错的。所有文字（包括树状图里的"分组成 / 排顺序 / 再感觉 / 感觉三步法 写啥都不怕"）必须由我们后期用 PS / Figma 叠加，不能依赖 AI 渲染。
